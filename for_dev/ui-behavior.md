# UI と挙動

WinLive の UI は、Windows 11 の PC 画面で邪魔になりにくいライブ通知を目指しています。

## 通常表示

- 小さな透明ピルとして表示
- タスクバーには表示しない
- 常に最前面
- 表示する activity がない場合は非表示
- タスクトレイには常駐

## 展開

展開時は 1 枚の大きなパネルにはせず、同じサイズのタイルを積みます。
これにより、横方向の見切れを避けやすくしています。

展開方向:

- `Up`: 現在の島を下に保ち、追加タイルを上に積む
- `Down`: 現在の島から下へ追加タイルを積む

設定画面で `Up` / `Down` を選択できます。

## ドラッグ移動

島本体をドラッグすると位置を変更できます。
ボタンを押した場合はドラッグではなくボタン操作を優先します。

展開中にドラッグした場合でも、保存される位置は通常サイズの島を基準にします。
そのため、次回起動時や折りたたみ時の位置が不自然になりにくいです。

## アニメーション

- 表示時: フェード、軽いスケール、スライド
- 展開時: 追加タイルをフェードとスライドで表示
- 閉じる時: 通常サイズへ戻す

v1 では軽量さを優先し、複雑な物理アニメーションは入れていません。
