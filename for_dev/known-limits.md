# 既知の制限

## タスクバー進捗

任意のアプリが `ITaskbarList3` で出したタスクバー進捗を、別アプリから安定して読み取る公式 API は前提にしていません。

安定ルートは WinLive の localhost API です。

## UI Automation 進捗検出

UI Automation による既存アプリの ProgressBar 探索は実験機能です。
既定では無効で、設定画面から有効化します。

制限:

- アプリによって検出できない
- 表示名や構造に依存する
- 誤検出の可能性がある
- ウィンドウタイトルではなく ProgressBar の AutomationId / 名前 / クラス名を優先して ID を作るが、アプリ側の実装によっては安定しない

## 設定反映

次の変更は保存後、WinLive の再起動が必要になる場合があります。

- API サーバーの有効 / 無効
- API ポート
- API トークン
- 実験的な進捗検出

## 配置

初期位置は、一般的な Windows 11 の下タスクバーと通知領域付近を主対象にしています。
複数モニターや特殊なタスクバー配置では、追加の調整が必要になる可能性があります。

## 署名

未署名の exe は、Windows SmartScreen の警告対象になる場合があります。
公開配布を広げる場合はコード署名を検討してください。
